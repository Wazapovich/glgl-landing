# [gl.gl] — Good luck is built, not given.

Landing page for GLGL — the habit tracker for people who play the long game.

## Stack
- Pure HTML/CSS/JS (no framework needed)
- Deployed on Cloudflare Pages
- Domain: [glgl.app](https://glgl.app)

## Deploy
Push to `main` branch → auto-deploys to Cloudflare Pages.

## Local dev
Just open `index.html` in your browser. No build step needed.
